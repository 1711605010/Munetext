### 成员：梁伟浩、孙小蝶

### 2019-06-17 周一完成内容：
#### 1、需求分析：本次课程实现一款满足用户的需求菜谱APP，其中包括菜谱搜索功能，菜谱分类功能。用户尝试做新菜时需要查找菜谱，有强大的菜谱搜索功能，搜出来的菜谱靠谱。针对用户的这种需求，菜谱APP在设计上满足受众搜索菜谱名、食材名。
